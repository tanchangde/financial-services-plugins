# 投行术语参考

## 随意到专业的替换

| 随意/非正式 | 投行标准 |
|-------------|----------|
| "a lot of growth" | "显著增长" 或 "X%增长" |
| "pretty good margins" | "有吸引力的利润率" 或 "X%利润率" |
| "they bought the company" | "该公司被收购" |
| "big deal" | "变革性交易" |
| "cheap valuation" | "有吸引力的估值" 或 "估值折价" |
| "expensive" | "溢价估值" |
| "make more money" | "提升盈利能力" 或 "推动利润率扩张" |
| "getting bigger" | "追求增长" 或 "扩展运营" |
| "cut costs" | "实施成本优化" 或 "推动运营效率" |
| "good fit" | "战略契合" 或 "引人注目的战略理由" |
| "help with" | "支持" 或 "促进" |
| "a bunch of" | "多个" 或 "众多" |
| "kind of" / "sort of" | [删除或具体说明] |
| "really" / "very" | [删除或量化] |
| "tons of" | "大量" 或量化 |
| "huge" | "显著" 或量化 |
| "pretty much" | [删除或精确说明] |
| "basically" | [删除或澄清] |

## 应避免的语言模式

- **缩写**：Don't → Do not，won't → will not
- **感叹号**：通常不适用于投行材料
- **第一人称**："We think..." → "管理层认为..." 或使用被动语态
- **无证据的最高级**："best-in-class" 需要支持数据
- **模糊量词**："some"、"many"、"several" → 具体数字

## 首选措辞模式

**增长叙述**：
- "展示了X%收入CAGR的业绩记录"
- "在[期间]内利润率持续扩张"
- "经证实的有机增长能力"

**市场地位**：
- "[特定细分市场]第X大参与者"
- "[特定产品]的领先提供商"
- "通过[特定属性]实现差异化定位"

**战略理由**：
- "引人注目的战略契合，源于..."
- "通过...创造有吸引力的价值机会"
- "来自[具体来源]的X百万美元协同潜力"
