# 可比公司分析

## ⚠️ 关键：数据源优先级（先读）

**始终遵循此数据源层级：**

1. **首先：检查MCP数据源** - 如果S&P Kensho MCP、FactSet MCP或Daloopa MCP可用，专门使用它们获取财务和交易信息
2. **如果MCP数据源可用，不要使用网络搜索**
3. **只有MCP不可用时：** 然后使用彭博终端、SEC EDGAR申报或其他机构来源
4. **永远不要将网络搜索作为主要数据源** - 它缺乏机构级分析所需的准确性、审计跟踪和可靠性

**为什么这很重要：** MCP来源提供经过验证的机构级数据，带有适当的引用。网络搜索结果可能过时、不准确或不可靠。

---

## 概述
此技能教Claude构建机构级可比公司分析，结合运营指标、估值倍数和统计基准测试。输出是一个结构化的Excel/电子表格，通过同行比较支持明智的投资决策。

**参考材料和背景说明：**

在`examples/comps_example.xlsx`中提供了一个可比公司分析示例。在使用此技能或此技能目录中的其他示例文件时，智能地使用它们：

**用于：**
- 理解结构层次（部分如何流动）
- 掌握预期严谨程度（统计深度、文档标准）
- 学习原则（清晰的标题、透明的公式、审计跟踪）

**不要用于：**
- 精确复制格式或指标
- 不考虑上下文地复制布局
- 无论受众如何都应用相同的视觉风格

**首先始终问自己：**
1. **"您有首选格式还是我应该调整模板风格？"**
2. **"受众是谁？"**（投资委员会、董事会演示、快速参考、详细备忘录）
3. **"关键问题是什么？"**（估值、增长分析、竞争定位、效率）
4. **"背景是什么？"**（并购评估、投资决策、板块基准测试、绩效回顾）

**根据具体情况进行调整：**
- **行业背景**：大型科技超级巨头与新兴SaaS初创公司需要不同的指标
- **板块特定需求**：尽早添加相关指标（例如，科技的云ARR、企业客户、开发者生态系统）
- **公司熟悉度**：知名公司可能需要较少背景，更多关注增量分析
- **决策类型**：并购需要与持续投资组合监控不同的重点

**核心原则：** 使用模板原则（清晰结构、统计严谨、透明度公式）但根据背景调整执行。目标是机构级质量分析，而不是机构级外观模板。

用户提供的示例和明确偏好始终优先于默认值。

## 核心哲学
**"首先构建正确的结构，然后让数据说话。"**

从迫使战略性思考重要内容的标题开始，输入干净的数据，构建透明的公式，让统计数据自动出现。一份好的可比分析应让未参与构建的人也能立即理解。

## 第一部分：文档结构和设置

### 标题块（第1-3行）
```
第1行：[分析标题] - 可比公司分析
第2行：[公司列表及股票代码] • [公司1 (代码1)] • [公司2 (代码2)] • [公司3 (代码3)]
第3行：截至[期间] | 除每股金额和比率外，所有数字单位为[百万/十亿美元]
```

**为什么这很重要：** 立即建立背景。任何打开此文件的人都知道他们在看什么、创建时间以及如何解释数字。

### 视觉约定标准（可选 - 用户偏好和上传的模板始终优先）

**重要：这些只是建议的默认值。始终优先：**
1. 用户明确的格式偏好
2. 任何上传的模板文件格式
3. 公司/团队样式指南
4. 这些默认值（仅在没有其他指导时）

**建议的字体和排版：**
- **字体族**：Times New Roman（专业、可读、行业标准）
- **字体大小**：数据单元格11pt，标题12pt
- **粗体文本**：部分标题、公司名称、统计标签

**建议的颜色和阴影：**
- **部分标题**（例如"运营统计和财务指标"）：
  - 深蓝色背景（#17365D或类似海军蓝）
  - 白色粗体文本
  - 跨所有列全行阴影
- **列标题**（例如"公司"、"收入"、"利润率"）：
  - 浅蓝色/灰色背景（#D9E2F3或类似浅蓝）
  - 黑色粗体文本
  - 居中对齐
- **数据行**：
  - 公司数据白色背景
  - 黑色文本用于输入和公式
- **统计行**（最大值、75分位等）：
  - 浅灰色背景（#F2F2F2）
  - 黑色文本，左对齐标签

**建议的格式约定：**
- **小数精度**：
  - 百分比：1位小数（12.3%）
  - 倍数：1位小数（13.5x）
  - 美元金额：无小数，千位分隔符（69,632）
  - 显示为百分比的利润率：1位小数（68.7%）
- **边框**：无边框（干净、最小外观）
- **对齐**：所有指标居中对齐，以获得干净、统一的外观
- **单元格尺寸**：所有列宽应均匀/偶数，所有行高应一致（创造干净、专业的网格）

**注意：** 如果用户提供模板文件或指定不同格式，使用该格式。

## 第二部分：运营统计和财务指标

### 核心列（从这些开始）
1. **公司** - 格式一致的名称
2. **收入** - 规模指标（可根据上下文为LTM、季度或年度）
3. **收入增长率** - 同比百分比变化
4. **毛利润** - 收入减去销售成本
5. **毛利率** - 毛利润/收入（基本盈利能力）
6. **EBITDA** - 利息、税费、折旧、摊销前利润
7. **EBITDA利润率** - EBITDA/收入（运营效率）

### 可选添加（根据行业/目的选择）
- **季度vs LTM** - 如果季节性重要，两者都包括
- **自由现金流** - 对于资本密集型或SaaS企业
- **自由现金流利润率** - 自由现金流/收入（现金生成效率）
- **净利润** - 对于成熟、盈利的公司
- **营业利润** - 对于折旧摊销不同的企业
- **资本支出指标** - 对于资产密集型行业
- **40法则** - 专门针对SaaS（增长率% + 利润率%）
- **自由现金流转换** - 对于盈利质量分析（高级）

### 公式示例（使用第7行作为示例）
```excel
// 核心比率 - 这些始终计算
毛利率 (F7): =E7/C7
EBITDA利润率 (H7): =G7/C7

// 可选比率 - 如相关则包括
自由现金流利润率: =[自由现金流]/[收入]
净利润率: =[净利润]/[收入]
40法则: =[增长率 %]+[自由现金流利润率 %]
```

**黄金法则：** 每个比率应该是[某物]/[收入]或[某物]/[本表中的某物]。保持简单。

### 统计块（在公司数据之后）

**关键：为所有可比指标（比率、利润率、增长率、倍数）添加统计公式。**

```
[留一行空白以便视觉分隔]
- 最大值: =MAX(B7:B9)
- 75分位: =QUARTILE(B7:B9,3)
- 中位数: =MEDIAN(B7:B9)
- 25分位: =QUARTILE(B7:B9,1)
- 最小值: =MIN(B7:B9)
```

**需要统计的列：**
- 收入增长率%、毛利率%、EBITDA利润率%、每股收益
- 企业价值/收入、企业价值/EBITDA、市盈率、股息收益率%、贝塔系数

**不需要统计的列：**
- 收入、EBITDA、净利润（绝对规模因公司规模而异）
- 市值、企业价值（不同规模公司不可比）

**注意：** 在公司数据和统计行之间留一行空白以便视觉分隔。不要添加"板块统计"或"估值统计"标题行。

**为什么分位数很重要：** 它们显示分布，而不仅仅是平均值。75分位倍数告诉您"优质"公司以什么价格交易。

## 第三部分：估值倍数和投资指标

### 核心估值列（从这些开始）
1. **公司** - 与运营部分相同的顺序
2. **市值** - 当前市场估值
3. **企业价值** - 市值 ± 净债务/现金
4. **企业价值/收入** - 市场为每美元销售支付多少
5. **企业价值/EBITDA** - 市场为每美元盈利支付多少
6. **市盈率** - 价格相对净收益

### 可选估值指标（根据上下文选择）
- **自由现金流收益率** - 自由现金流/市值（用于现金重点分析）
- **PEG比率** - 市盈率/增长率（用于成长型公司）
- **市净率** - 市场价值与账面价值（用于资产密集型企业）
- **ROE/ROA** - 回报指标（用于盈利能力比较）
- **收入/EBITDA CAGR** - 历史增长率（用于趋势分析）
- **资产周转率** - 收入/资产（用于运营效率）
- **债务/权益** - 杠杆（用于资本结构分析）

**关键原则：** 包括对您的行业重要的3-5个核心倍数。不要仅仅因为可以就包括每个可能的指标。

### 公式示例
```excel
// 核心倍数 - 始终包括这些
企业价值/收入: =[企业价值]/[LTM收入]
企业价值/EBITDA: =[企业价值]/[LTM EBITDA]
市盈率: =[市值]/[净利润]

// 可选倍数 - 如有数据则包括
自由现金流收益率: =[LTM自由现金流]/[市值]
PEG比率: =[市盈率]/[增长率 %]
```

### 交叉引用规则
**关键：** 估值倍数必须引用运营指标部分。永远不要输入相同的原始数据两次。如果收入在C7，则企业价值/收入公式应引用C7。

### 统计块
与运营部分相同的结构：Max、75分位、中位数、25分位、最小值用于每个指标。在公司和统计数据之间留一行空白。不要添加"估值统计"标题行。

## 第四部分：附注和方法论文档

### 必需组件

**数据来源和质量：**
- 数据来自哪里？（S&P Kensho MCP、FactSet MCP、Daloopa MCP、彭博、SEC申报）
- 覆盖什么期间？（2024年第四季度，审计数字）
- 如何验证的？（与10-K/10-Q交叉核对）
- 注意：如果MCP数据源可用，优先使用它们以获得更好的准确性和可追溯性

**关键定义：**
- EBITDA计算方法（毛利润 + 折旧摊销，或营业利润 + 折旧摊销）
- 自由现金流公式（经营现金流 - 资本支出）
- 特殊指标说明（40法则、自由现金流转换）
- 期间定义（LTM、CAGR计算期间）

**估值方法论：**
- 企业价值如何计算？（市值 + 净债务）
- 使用了什么增长率？（历史CAGR、前瞻估计）
- 做了什么调整？（排除一次性项目、标准化利润率）

**分析框架：**
- 投资论点是什么？（云/SaaS效率）
- 什么指标最重要？（现金生成、资本效率）
- 读者应如何解释统计数据？（分位数提供背景）

## 第五部分：选择正确的指标（决策框架）

### 从"我回答什么问题开始？"

**"哪家公司被低估了？"**
→ 关注：企业价值/收入、企业价值/EBITDA、市盈率、市值
→ 跳过：运营细节、增长指标

**"哪家公司效率最高？"**
→ 关注：毛利率、EBITDA利润率、自由现金流利润率、资产周转率
→ 跳过：规模指标、绝对美元金额

**"哪家公司增长最快？"**
→ 关注：收入增长率%、EBITDA CAGR、用户/客户增长率
→ 跳过：利润率指标、杠杆比率

**"哪家公司现金生成最好？"**
→ 关注：自由现金流、自由现金流利润率、自由现金流转换、资本支出强度
→ 跳过：EBITDA、市盈率比率

### 行业特定指标选择

**软件/SaaS：**
必须有：收入增长率、毛利率、40法则
可选：ARR、净美元留存、CAC回收期
跳过：资产周转率、库存指标

**制造/工业：**
必须有：EBITDA利润率、资产周转率、资本支出/收入
可选：ROA、库存周转、积压订单
跳过：40法则、SaaS指标

**金融服务：**
必须有：ROE、ROA、效率比率、市盈率
可选：净利息margin、贷款损失准备金
跳过：毛利率、EBITDA（对银行无意义）

**零售/电商：**
必须有：收入增长率、毛利率、库存周转
可选：同店销售、客户获取成本
跳过：重资本支出或研发指标

### "5-10法则"

**5个运营指标** - 收入、增长、2-3个利润率/效率指标
**5个估值指标** - 市值、企业价值、3个倍数
**= 10列** - 足以讲故事，但不是太多以至于失去线索

如果超过15个指标，可能包含了噪音。无情地编辑。

## 第六部分：最佳实践和质量检查

### 开始之前
1. **确定同行群体** - 公司必须具有真正的可比性（相似的商业模式、规模和地理区域）
2. **选择正确的期间** - LTM平滑季节性；季度显示趋势
3. **提前标准化单位** - 百万vs十亿决定一切
4. **映射数据来源** - 知道每个数字来自哪里

### 构建时
1. **首先输入所有原始数据** - 在写公式之前完成蓝色文本
2. **为所有硬编码输入添加单元格注释** - 右键单元格 → 插入注释 → 记录来源或假设

   **对于来源数据，精确引用来源：**
   - 示例："彭博终端 - MSFT Equity DES，访问于2024-10-02"
   - 示例："2024年第四季度10-K申报，第42页，行项目'总收入'"
   - 示例："FactSet共识估计，截至2024-10-02"
   - **尽可能包含超链接**：右键单元格 → 链接 → 粘贴SEC申报、数据源或报告的URL

   **对于假设，解释推理：**
   - 示例："假设15% EBITDA利润率基于同行中位数，公司未披露"
   - 示例："估计企业价值为市值 + 5000万美元净债务（来自第三季度资产负债表，第四季度尚未发布）"
   - 示例："前瞻市盈率基于12位分析师平均共识EPS为3.45美元"

   **为什么这很重要**：启用审计跟踪、数据验证、假设透明度和未来更新
3. **逐行构建公式** - 在继续之前测试每个计算
4. **对标题使用绝对引用** - $C$6锁定标题行
5. **保持格式一致** - 百分比格式为百分比，不是小数
6. **添加条件格式** - 自动突出异常值

### 合理性检查
- **利润率测试**：毛利率 > EBITDA利润率 > 净利润率（根据定义总是成立）
- **倍数合理性**：
  - 企业价值/收入：通常0.5-20倍（因行业差异很大）
  - 企业价值/EBITDA：通常8-25倍（跨行业相当一致）
  - 市盈率：通常10-50倍（取决于增长率）
- **增长-倍数相关性**：更高增长通常意味着更高倍数
- **规模-效率权衡**：大公司通常有更好利润率（规模优势）

### 要避免的常见错误
❌ 在公式中混合市值和企业价值
❌ 对分子和分母使用不同时期（LTM vs 季度）
❌ 将硬编码数字而非单元格引用放入公式
❌ **硬编码输入没有单元格注释引用来源或解释假设**
❌ 缺少到SEC申报或数据源的超链接（如有）
❌ 包含太多指标没有明确目的
❌ 包含不可比的公司（不同商业模式）
❌ 使用未披露的过时数据
❌ 错误计算百分比的平均值（应该是中位数）

## 第七部分：高级功能

### 动态标题
对于显示计算的列，使用清晰的单位标签：
```
收入增长率（同比增长）% | EBITDA利润率 | 自由现金流利润率 | 40法则
```

### 分位数分析
不仅仅是中位数，分位数显示：
- **75分位** = "优质"公司在此交易
- **中位数** = 典型市场估值
- **25分位** = "折扣"领域

这有助于回答："我们的目标公司相对于同行是贵还是便宜？"

### 行业特定修改

**软件/SaaS：**
- 添加：ARR、净美元留存、CAC回收期
- 强调：40法则、自由现金流利润率、毛利率>70%

**医疗保健：**
- 添加：研发/收入、管道价值、监管状态
- 强调：EBITDA利润率、增长率、报销风险

**工业：**
- 添加：积压、订单趋势、地理分布
- 强调：ROIC、资产周转率、周期调整

**消费品：**
- 添加：同店销售、客户获取成本、品牌价值
- 强调：收入增长率、毛利率、库存周转

## 第八部分：工作流程和实用技巧

### 逐步流程
1. **设置结构**（30分钟）
   - 创建所有标题
   - 格式化单元格（蓝色输入，黑色公式）
   - 锁定单位和日期引用

2. **收集数据**（60-90分钟）
   - 从主要来源拉取（如果可用：S&P Kensho MCP、FactSet MCP、Daloopa MCP；否则彭博、SEC）
   - 用蓝色输入所有原始数字
   - 在附注部分记录来源

3. **构建公式**（30分钟）
   - 从简单比率开始（利润率）
   - 进展到倍数（企业价值/收入）
   - 添加交叉检查（利润率合理吗？）

4. **添加统计**（15分钟）
   - 为所有列复制公式结构
   - 验证范围正确（B7:B9，不是B7:B10）
   - 检查分位数逻辑

5. **质量控制**（30分钟）
   - 运行合理性检查
   - 验证公式引用
   - 检查#DIV/0!或#REF!错误
   - 与已知基准比较

6. **文档**（15分钟）
   - 完成附注部分
   - 添加数据来源
   - 定义方法论
   - 标注分析日期

- **保存模板**：构建一次，永远重用
- **颜色编码异常值**：条件格式对于>2个标准差的值

- **版本控制**：保存为"Comps_v1_2024-12-15"，日期清晰

## 第十一部分：红旗和警告信号

### 数据质量问题
🚩 不一致的时区（混合季度和年度）
🚩 缺失数据无解释
🚩 数据来源之间重大差异（>10%方差）

### 估值红旗
🚩 负EBITDA公司用EBITDA倍数估值（改用收入倍数）
🚩 市盈率>100倍没有超增长故事
🚩 利润率对该行业没有意义

### 可比性问题
🚩 不同财年结束（导致时间问题）
🚩 混合单一业务公司和大型企业集团（不应混用）
🚩 标记为"可比公司"但商业模式实质不同

**如有疑问，排除该公司。** 3个完美的可比公司好于6个可疑的。

## 第十二部分：公式参考指南

### 基本Excel公式
```excel
// 统计函数
=AVERAGE(range)          // 简单平均值
=MEDIAN(range)           // 中间值
=QUARTILE(range, 1)      // 25分位
=QUARTILE(range, 3)      // 75分位
=MAX(range)              // 最大值
=MIN(range)              // 最小值
=STDEV.P(range)          // 标准差

// 财务计算
=B7/C7                   // 简单比率（利润率）
=SUM(B7:B9)/3            // 多公司平均值
=IF(B7>0, C7/B7, "N/A")  // 条件计算
=IFERROR(C7/D7, 0)       // 处理除零

// 跨表引用
='Sheet1'!B7             // 引用另一个表
=VLOOKUP(A7, Table1, 2)  // 从数据表查找
=INDEX(MATCH())          // 高级查找

// 格式
=TEXT(B7, "0.0%")        // 格式化为百分比
=TEXT(C7, "#,##0")       // 千位分隔符
```

### 常见比率公式
```excel
毛利率 = 毛利润 / 收入
EBITDA利润率 = EBITDA / 收入
自由现金流利润率 = 自由现金流 / 收入
自由现金流转换 = 自由现金流 / 经营现金流
ROE = 净利润 / 股东权益
ROA = 净利润 / 总资产
资产周转率 = 收入 / 总资产
债务/权益 = 总债务 / 股东权益
```

## 关键原则总结

1. **结构驱动洞察** - 正确的标题迫使正确思考
2. **少即是多** - 重要的5-10个指标胜过不重要的20个
3. **为问题选择指标** - 估值分析≠效率分析
4. **统计显示模式** - 中位数/分位数比平均值揭示更多
5. **透明度胜过复杂** - 简单公式每个人都能理解
6. **可比性为王** - 排除比强迫一个糟糕的可比公司更好
7. **记录您的选择** - 在附注部分解释哪些指标以及为什么
