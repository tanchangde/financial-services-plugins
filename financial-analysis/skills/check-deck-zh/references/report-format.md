# 演示文稿检查报告格式

## 报告模板

```markdown
# 演示文稿检查报告：[演示文稿名称]

## 摘要
- 问题总数：X
- 关键：X（数字不匹配、事实错误）
- 重要：X（叙事数据对齐、语言）
- 次要：X（格式）

## 关键问题

### 数据一致性
1. **[问题名称]**（第X、Y页）
   - 第X页：[值]
   - 第Y页：[值]
   - 操作：[建议]

### 数据与叙事对齐
1. **[问题名称]**（第X、Y页）
   - 声明："[引用文本]"
   - 数据显示：[矛盾]
   - 操作：[建议]

## 重要问题

### 语言润色
1. **[问题类型]**（第X页）
   - 当前："[引用文本]"
   - 建议："[替换]"

## 次要问题

### 格式
1. **[问题类型]**（第X页）
   - [描述和修复]

## 最终检查清单
- [ ] 数字已核对
- [ ] 叙事与数据匹配
- [ ] 语言符合投行标准
- [ ] 图表已标注来源
- [ ] 格式一致
```

## 问题严重程度分类

**关键**（客户交付前必须修复）：
- 幻灯片间数字不匹配
- 计算错误
- 事实不准确（姓名、职务、日期）
- 数据与叙事矛盾

**重要**（应该修复）：
- 随意/非正式语言
- 无具体性的模糊声明
- 术语不一致
- 缺少图表来源

**次要**（润色项）：
- 字体/颜色不一致
- 日期格式变化
- 间距/对齐问题
- 孤立文字
