# 格式标准参考

| 元素 | 格式 |
|------|------|
| 硬编码输入 | 蓝色字体 |
| 公式 | 黑色字体 |
| 链接到其他工作表 | 绿色字体 |
| 检查单元格 | 错误时为红色，平衡时为绿色 |
| 负值 | 使用括号，而不是负号 |
| 货币 | 大额数字无小数，每股数据2位小数 |
| 百分比 | 1位小数 |
| 标题 | 粗体，底部边框 |
| 单位行 | 包含单位行（百万美元、%等） |

## 视觉分隔指南

- 历史列和预测列之间使用细竖边框
- 部分总计（如资产总计）后使用粗底边框
- 小计使用单底边框
- 合计使用双底边框

## 合计和小计行格式

所有合计和小计行必须对其数值使用**粗体字体格式**，以清楚区分汇总数字与单个行项目。

## 资产负债表检查行格式

资产负债表检查行（在负债和权益合计下方）使用条件数字格式，将非零值显示为红色。当资产负债表正确平衡时（检查 = 0），值以黑色或标准格式显示。

| 检查值 | 字体颜色 |
|--------|----------|
| = 0（平衡） | 黑色（标准） |
| ≠ 0（错误） | 红色 |

**实现：**应用自定义数字格式`[Red][<>0]0.00;[Red][<>0](0.00);0.00`，或使用Excel条件格式，规则为"单元格值 ≠ 0" → 红色字体。

## 利润率行格式

| 元素 | 格式 |
|------|------|
| 利润率行 | 缩进，斜体，1位小数 |
| 正向趋势 | 无特殊格式（或淡绿色） |
| 负向趋势 | 标记以供审查（淡黄色） |
| 低于同行平均 | 考虑突出显示以供讨论 |

## 信贷指标格式

| 元素 | 格式 |
|------|------|
| 杠杆倍数 | 1位小数，带"x"后缀（例如2.5x） |
| 百分比 | 1位小数，带"%"后缀 |
| 净债务为负 | 括号，表示净现金状况 |
| 部分标题 | 粗体，"信贷指标" |
| 分隔线 | 信贷指标部分上方细边框 |

## 信贷指标阈值颜色

| 指标 | 绿色 | 黄色 | 红色 |
|------|------|------|------|
| 总债务 / EBITDA | < 2.5x | 2.5x-4.0x | > 4.0x |
| 净债务 / EBITDA | < 2.0x | 2.0x-3.5x | > 3.5x |
| 利息覆盖率 | > 4.0x | 2.5x-4.0x | < 2.5x |
| 债务 / 总资本 | < 40% | 40%-60% | > 60% |
| 流动比率 | > 1.5x | 1.0x-1.5x | < 1.0x |
| 速动比率 | > 1.0x | 0.75x-1.0x | < 0.75x |

## 检查标签页的条件格式

- 单元格包含通过指示符 → 绿色填充
- 单元格包含失败指示符 → 红色填充
- 单元格包含警告 → 黄色填充
- 差异单元格 = 0 → 浅绿色填充
- 差异单元格 ≠ 0 → 浅红色填充

## 利润率合理性标志

- 毛利率 < 0% → 错误：审查销售成本
- 毛利率 > 80% → 警告：验证收入/销售成本
- EBITDA利润率 < 0% → 标记：经营亏损
- EBITDA利润率 > 50% → 警告：异常高
- 净利润率 < 0% → 标记：净亏损（增长阶段可能可接受）
- 净利润率 > 毛利率 → 错误：公式问题
