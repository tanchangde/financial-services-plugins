# SEC申报文件数据提取参考

**何时使用：** 仅在模型模板特别要求从SEC申报文件（10-K、10-Q）提取数据时参考此文件。对于直接提供数据或使用其他数据源的模板，不需要此参考。

---

## 从SEC申报文件提取数据（10-K / 10-Q）

用公开公司数据填充模型模板时，直接从SEC申报文件提取财务数据。

### 第一步：查找申报文件

1. 使用SEC EDGAR：`https://www.sec.gov/cgi-bin/browse-edgar?action=getcompany&CIK=[TICKER]&type=10-K`
2. 对于季度数据，使用`type=10-Q`

### 第二步：识别申报货币

提取数据之前，识别报告货币：
- 查看封面页或标题上的报告货币
- 查看报表标题（例如"以千美元为单位"）
- 查看附注1（重要会计政策摘要）

**常见货币指示符**

| 指示符 | 货币 |
|--------|------|
| $, USD | 美元 |
| €, EUR | 欧元 |
| £, GBP | 英镑 |
| ¥, JPY | 日元 |
| ¥, CNY, RMB | 人民币 |
| CHF | 瑞士法郎 |
| CAD, C$ | 加拿大元 |

将模型货币设置为与申报文件匹配；在假设标签页中记录。

### 第三步：导航至财务报表

在10-K或10-Q中查找：
- **项目8**（10-K）或**项目1**（10-Q）：财务报表
- 要提取的关键部分：
  - 合并利润表（损益表）
  - 合并资产负债表
  - 合并现金流量表
  - 财务报表附注（用于明细表详情）

### 第四步：数据提取映射

**损益表（来自合并利润表）**

| 申报文件行项目 | 模型行项目 |
|----------------|------------|
| 净收入/净销售额 | 收入 |
| 销售成本 | 销售成本 |
| 销售、一般和行政 | 销售及管理费用 |
| 折旧和摊销 | 折旧摊销 |
| 利息费用净额 | 利息费用 |
| 所得税费用 | 税费 |
| 净利润 | 净利润 |

**资产负债表（来自合并资产负债表）**

| 申报文件行项目 | 模型行项目 |
|----------------|------------|
| 现金及现金等价物 | 现金 |
| 应收账款净额 | 应收账款 |
| 存货 | 存货 |
| 固定资产净额 | 固定资产（净额） |
| 资产总计 | 资产总计 |
| 应付账款 | 应付账款 |
| 短期债务/长期债务流动部分 | 流动债务 |
| 长期债务 | 长期债务 |
| 留存收益 | 留存收益 |
| 股东权益合计 | 权益合计 |

**现金流量表（来自合并现金流量表）**

| 申报文件行项目 | 模型行项目 |
|----------------|------------|
| 净利润 | 净利润 |
| 折旧和摊销 | 折旧摊销 |
| 应收账款变化 | Δ应收账款 |
| 存货变化 | Δ存货 |
| 应付账款变化 | Δ应付账款 |
| 资本支出 | 资本支出 |
| 发行普通股收益 | 权益发行 |
| 债务发行/偿还收益 | 债务活动 |
| 支付的股息 | 股息 |

### 第五步：从附注提取支持明细

对于明细表，从财务报表附注提取：
- **附注：债务** → 到期计划、利率、契约
- **附注：固定资产** → 固定资产原值、累计折旧、使用年限
- **附注：收入** → 分部明细、地理拆分
- **附注：租赁** → 经营与融资租赁义务

### 第六步：历史数据要求

至少提取3年的历史数据：
- 10-K提供3年的IS/CF，2年的BS
- 对于第3年的BS，从前一年的10-K提取
- 如需要，使用10-Q填充季度细分

### 数据提取检查清单

- 识别报告货币和比例（千、百万）
- 3年历史损益表
- 3年历史现金流量表
- 3年历史资产负债表
- 验证IS净利润 = CF期初净利润（每年）
- 验证BS现金 = CF期末现金（每年）
- 从附注提取债务到期计划
- 提取折旧摊销明细或使用年限假设
- 注意任何非经常性/一次性项目以便标准化

### 处理常见的申报差异

| 差异 | 如何处理 |
|------|----------|
| 折旧摊销包含在销售成本/销售及管理费用中 | 从现金流量表提取折旧摊销 |
| "其他"行项目重大 | 查看附注分解 |
| 重述 | 使用重述数字，在假设中注明 |
| 财年 ≠ 日历年 | 使用财年结束日期标记（例如FYE 2025年1月） |
| 非美元报告货币 | 调整模型货币以匹配申报 |
