# 公式参考

**重要：**除非用户另有说明，否则使用本参考文件中概述的公式。

---

## 核心链接

```
资产负债表：        资产 = 负债 + 权益
净利润：           IS净利润 → CF经营活动（起点）
现金流量：          Δ现金 = CFO + CFI + CFF
现金勾稽：         期末现金（CF）= 现金（BS资产）
现金月度/年度：    期末现金（月度）= 期末现金（年度）
留存收益：         前期RE + 净利润 - 股息 = 期末RE
权益发行：         Δ普通股/额外实收资本（BS）= 权益发行（CFF）
第0年权益：        第0年募集的权益 = 第1年期初权益
```

## 毛利润计算

**重要：**毛利润必须从净收入计算，而不是总收入。

```
净收入 - 营业成本 = 毛利润
```

| 术语 | 定义 |
|------|------|
| 总收入 | 任何扣除前的总收入 |
| 净收入 | 总收入 - 退货 - 津贴 - 折扣 |
| 营业成本 | 生产所售商品/服务的直接成本 |
| 毛利润 | 净收入 - 营业成本 |

**注意：**始终使用净收入（也称为"净销售额"或大多数财务报表上的"收入"）作为盈利能力计算的起点。总收入会夸大真实的顶部表现。

## 利润率公式

```
毛利率 %      = 毛利润 / 净收入
EBITDA        = EBIT + 折旧摊销  （或 = 毛利润 - 运营费用）
EBITDA利润率 % = EBITDA / 净收入
EBIT利润率 %   = EBIT / 净收入
净利润率 %    = 净利润 / 净收入
```

## 信贷指标公式

```
总债务            = 流动债务部分 + 长期债务
净债务            = 总债务 - 现金
总债务 / EBITDA   = 总债务 / EBITDA（来自IS）
净债务 / EBITDA   = 净债务 / EBITDA（来自IS）
利息覆盖率        = EBITDA / 利息费用（来自IS）
净利息费用 % 债务 = 净利息费用 / 长期债务
债务 / 总资本     = 总债务 /（总债务 + 总权益）
债务 / 权益       = 总债务 / 总权益
流动比率          = 流动资产 / 流动负债
速动比率          = （流动资产 - 存货）/ 流动负债
```

## 预测公式（净收入百分比法）

```
营业成本（预测） = 净收入 × 营业成本 % 假设
销售及管理费用（预测） = 净收入 × 销售及管理费用 % 假设
一般及行政费用（预测） = 净收入 × 一般及行政费用 % 假设
研发费用（预测） = 净收入 × 研发费用 % 假设
股票补偿（预测） = 净收入 × 股票补偿 % 假设
```

## 营运资本公式

```
应收账款
  前期应收账款
  + 收入（来自IS）
  - 现金收款（plug）
  = 期末应收账款
  DSO = （应收账款 / 收入）× 365

存货
  前期存货
  + 采购（plug）
  - 销售成本（来自IS）
  = 期末存货
  DIO = （存货 / 销售成本）× 365

应付账款
  前期应付账款
  + 采购（来自存货计算）
  - 现金付款（plug）
  = 期末应付账款
  DPO = （应付账款 / 销售成本）× 365

净营运资本 = 应收账款 + 存货 - 应付账款
ΔWC = 当前净营运资本 - 前期净营运资本
```

## 折旧摊销明细表公式

```
期初固定资产原值
+ 资本支出
= 期末固定资产原值

期初累计折旧
+ 折旧费用
= 期末累计折旧

固定资产净额 = 固定资产原值 - 累计折旧
```

## 债务明细表公式

```
期初债务余额
+ 新借款
- 偿还
= 期末债务余额

利息费用 = 平均债务余额 × 利率
  （使用期初余额以避免循环，或如果启用循环引用则迭代）
```

## 留存收益公式

```
期初留存收益
+ 净利润（来自IS）
+ 股票补偿（来自IS）
- 股息
= 期末留存收益
```

## 净经营亏损（NOL）明细表公式

```
NOL结转明细表

期初NOL余额（第1年/成立 = 0）
+ 生成的NOL（如果税前利润 < 0，则为ABS（税前利润），否则为0）
- 使用的NOL（受限于应税收入和利用上限）
= 期末NOL余额

起始余额规则

对于新业务或第一个建模期间：
  期初NOL余额 = 0
  NOL只能通过已实现亏损（税前利润 < 0）增加
  NOL不能无中生有或假设

NOL利用计算

税前利润
  如果税前利润 > 0：
    可用NOL = 期初NOL余额
    利用上限 = 税前利润 × 80%（2017年后的联邦限制）
    使用的NOL = MIN（可用NOL，利用上限）
    应税收入 = 税前利润 - 使用的NOL
  如果税前利润 ≤ 0：
    使用的NOL = 0
    应税收入 = 0
    生成的NOL = ABS（税前利润）

含NOL的税费计算

应缴税费 = MAX（0，应税收入 × 税率）
  （税费不能为负；亏损创建NOL资产而不是）

NOL的递延税资产（DTA）

DTA - NOL结转 = 期末NOL余额 × 税率
ΔDTA = 当前DTA - 前期DTA
  （DTA增加 = CF上的非现金收益）
  （DTA减少 = CF上的非现金费用）
```

## 资产负债表结构

```
资产
  现金（来自CF期末现金）
  应收账款（来自营运资本）
  存货（来自营运资本）
  流动资产合计

  固定资产净额（来自折旧摊销）
  递延税资产 - NOL（来自NOL明细表）
  非流动资产合计
  资产合计

负债
  应付账款（来自营运资本）
  债务流动部分（来自债务）
  流动负债合计

  长期债务（来自债务）
  负债合计

权益
  普通股
  留存收益（来自RE明细表）
  权益合计

检查：资产 - 负债 - 权益 = 0
```

## 现金流量表结构

```
经营活动产生的现金（CFO）
  净利润（链接：IS）
  + 折旧摊销（链接：折旧摊销明细表）
  + 股票补偿（链接：IS或假设）
  - ΔDTA（递延税资产）（链接：NOL明细表；DTA增加 = 现金使用）
  - Δ应收账款（链接：营运资本）
  - Δ存货（链接：营运资本）
  + Δ应付账款（链接：营运资本）
  = CFO

投资活动产生的现金（CFI）
  - 资本支出（链接：折旧摊销明细表）
  = CFI

融资活动产生的现金（CFF）
  + 债务发行（链接：债务）
  - 债务偿还（链接：债务）
  + 权益发行（链接：BS普通股/额外实收资本）
  - 股息（链接：RE明细表）
  = CFF

现金净变化 = CFO + CFI + CFF
期初现金
+ 现金净变化
= 期末现金（链接至：BS现金）
```

## 检查公式

```
BS平衡检查：       = 资产 - 负债 - 权益  （必须 = 0）
现金勾稽：         = BS现金 - CF期末现金       （必须 = 0）
RE滚动：           = 前期RE + 净利润 + 股票补偿 - 股息 - BS RE  （必须 = 0）
DTA勾稽：          = NOL明细表DTA - BS DTA      （必须 = 0）
权益发行勾稽：     = Δ普通股/额外实收资本（BS）- 权益发行（CFF）  （必须 = 0）
第0年权益勾稽：    = 第0年募集的权益 - 第1期初权益  （必须 = 0）
现金月度vs年度：   = 期末现金（月度）- 期末现金（年度）  （必须 = 0）
NOL利用上限：      = 使用的NOL ≤ 税前利润 × 80%       （对于2017年后必须为TRUE）
NOL非负：          = 期末NOL余额 ≥ 0         （必须为TRUE）
NOL期初余额：      = 期初NOL（第1年）= 0     （对于新业务必须为TRUE）
NOL累积：          = NOL仅在税前利润 < 0时增加（亏损产生NOL）
