# 项目挖掘（Deal Sourcing）

description: PE（私募股权）项目挖掘工作流程 — 发现目标公司、检查 CRM（客户关系管理）中的现有关系，并起草个性化的创始人外联邮件。适用于挖掘新项目、在某个行业 prospecting（ prospecting 指主动寻找潜在客户/目标）或联系创始人。触发词："find companies"、"source deals"、"draft founder email"、"check if we've seen this company"或"outreach to founder"。

## 工作流程

本技能遵循 3 步挖掘流程：

### 步骤 1：发现公司

根据用户的标准研究和识别潜在目标公司：

- **行业/领域重点**：询问用户正在关注什么领域（例如，"医疗行业的 B2B SaaS"、"东南地区的工业服务"）
- **交易参数**：收入范围、EBITDA（息税折旧摊销前利润）范围、增长概况、地理位置、所有权类型（创始人所有、PE 支持、企业资产剥离）
- **信息来源**：使用网络搜索查找符合标准的公司。查看行业报告、会议参会者名单、行业出版物和竞争格局
- **输出**：公司短名单，包括：名称、描述、预估收入/规模、所在地、创始人/CEO 姓名、网站，以及为何符合投资主题

### 步骤 2：CRM 检查

在联系之前，检查公司或创始人是否已存在于公司的 CRM 中：

- 在用户的邮箱（Gmail）中搜索与该公司或创始人的先前通信
- 在 Slack 中搜索任何内部提及或先前关于该目标的讨论
- 询问用户："您或您的团队之前是否与 [公司] 有过联系？"
- 标记任何现有关系、先前的否决或已知背景
- **输出**：对于每家公司，注明："新公司"（无先前联系）、"现有联系"（找到先前通信 — 总结）或"先前已否决"（如有先前否决的证据）

### 步骤 3：起草创始人外联邮件

起草给创始人/CEO 的个性化冷启动邮件：

- **语气**：专业但温暖。不要过于正式 — 创始人对真诚、简洁的联系反应更好
- **结构**：
  1. 简短介绍 — 您是谁以及您的公司（如未知则询问用户其公司介绍）
  2. 为什么这家公司引起了您的注意 — 引用具体事项（产品、市场地位、增长）
  3. 您在寻找什么 — 合作伙伴关系，而不仅仅是交易
  4. 温和的请求 — "您愿意进行一次简短的对话吗？"
- **个性化**：引用公司的具体产品、近期新闻或市场地位。切勿使用通用模板
- **长度**：最多 4-6 句话。创始人很忙
- **语气匹配**：如果用户之前发送过外联邮件，研究它们以匹配其语气和风格。在 Gmail 中搜索"已发送"邮件，关键词如"reaching out"、"introduction"、"partnership"以查找示例

### 邮件起草指南

- 主题行：保持简短具体。引用公司或行业，不要用"投资机会"
- 首次联系不要添加附件
- 包含清晰但低压力的 CTA（行动号召）
- 如有 Gmail 可用则在 Gmail 中起草，否则输出为文本供用户复制

## 示例交互

**用户**："为我找德克萨斯州收入在 1000-5000 万美元的创始人所有的工业服务公司"

**助手**：
1. 在网络搜索符合标准的德克萨斯州工业服务公司
2. 提供 5-8 家公司的短名单及关键详情
3. 对每家公司，检查 Gmail/Slack 是否有先前联系
4. 为标记为"新公司"的起草个性化外联邮件
5. 提交草稿供用户审查后再发送

## 重要提示

- 在起草邮件之前，始终提交短名单供用户审查
- 未经用户明确批准，切勿发送邮件
- 如果用户的公司介绍或投资标准不明确，在起草前询问
- 质量优于数量 — 5 个经过充分研究的目标胜过 20 个通用目标
