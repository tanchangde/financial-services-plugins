# 买方名单

## 工作流程

### 第一步：了解目标公司

- 公司描述、行业和商业模式
- 收入、EBITDA（息税折旧摊销前利润）和增长状况
- 关键资产和能力（知识产权、客户关系、地理覆盖、团队）
- 预期估值范围
- 卖方偏好（战略 vs. 财务、管理层连续性、时间表）

### 第二步：战略买家

识别各类战略收购方：

**直接竞争对手**
- 同一领域的公司将获得市场份额
- 理由：收入协同效应、消除竞争、实现规模经济

**相邻市场参与者**
- 相邻市场的公司可以扩展到目标公司领域
- 理由：产品延伸、交叉销售、新市场进入

**垂直整合者**
- 客户或供应商可以进行垂直整合
- 理由：供应链控制、利润率提升、战略绑定

**平台构建者**
- 通过并购在该领域构建平台的大型公司
- 理由：整合型收购、填补能力空白

对每个战略买家进行评估：

| 买家 | 行业 | 收入 | 战略匹配度 | 财务能力 | 并购业绩记录 | 可能性 | 优先级 |
|------|------|------|------------|----------|--------------|--------|--------|
| | | | 高/中/低 | | 活跃/中等/无 | | A/B/C |

### 第三步：财务赞助商

识别私募股权/财务买家：

**平台型基金投资者**
- 在该行业寻找新平台进行投资的赞助商
- 标准：基金规模、行业专注、交易规模范围

**追加型买家**
- 在现有投资组合公司中可以收购目标公司作为补强的赞助商
- 识别具体的投资组合公司和协同效应理由

**成长型股权**
- 适用于早期或高增长目标
- 少数股权 vs. 多数股权偏好

对每个赞助商：

| 赞助商 | 基金规模 | 行业专注 | 投资组合重叠 | 近期活动 | 优先级 |
|---------|-----------|-------------|-------------------|-----------------|----------|
| | | | | | A/B/C |

### 第四步：优先级排序

对买方名单进行梯队分层：

- **第一梯队（5-10 家）**：最高战略匹配度、有记录的收购方、明确理由 — 首先接触
- **第二梯队（10-15 家）**：匹配度好但不太明显 — 在第二轮接触
- **第三梯队（10-20 家）**：可能但概率较低 — 如果需要扩大流程范围再接触

### 第五步：联系人映射

对每个第一梯队买家：
- 关键决策者（CEO、企业开发负责人（Corp Dev）、合伙人）
- 关系状态（现有关系、陌生接触、需要介绍）
- 已知偏好或限制（规模、地理、结构）
- 最佳接触渠道

### 第六步：输出

- Excel 工作簿包含：
  - 战略买家工作表（按梯队排序）
  - 财务赞助商工作表（按梯队排序）
  - 第一梯队联系人映射
  - 汇总统计（按梯队、按类型的买家总数）
- 用于委托函或路演的买方名单摘要（一页）

## 重要说明

- 质量重于数量 — 一份聚焦的 30-40 家经过深入研究的买家名单胜过 200 家的名单
- 研究近期并购活动 — 刚在该领域完成交易的买家要么渴望更多，要么已耗尽
- 检查直接竞争对手的反垄断问题 — 标记任何可能面临监管问题的买家
- 财务赞助商：检查基金年份和部署节奏 — 接近投资期结束的基金可能更有动力
- 始终询问卖方是否有他们想要包含或排除的买家
- 随着流程进展更新名单 — 根据反馈调整买家所属梯队
