# 客户回顾会议准备

## 工作流程

### 步骤 1：客户背景

收集或查找：
- **客户姓名** 和家庭成员
- **账户类型**：应税账户、个人退休账户（IRA）、Roth 账户、401(k) 退休计划、信托等
- **管理资产规模（AUM）总计**
- **投资政策声明（IPS）**：目标配置、风险承受能力、限制条件
- **人生阶段**：积累期、退休前、退休期、遗产传承期
- **上次会议日期** 和任何未完成的行动项目

### 步骤 2：投资组合业绩

针对每个账户和家庭汇总：

| 指标 | 季度至今（QTD） | 年初至今（YTD） | 1 年 | 3 年 | 自成立以来 |
|--------|-----|-----|--------|--------|----------------|
| 投资组合回报 | | | | | |
| 基准回报 | | | | | |
| 阿尔法 | | | | | |

**业绩归因：**
- 哪些资产类别/头寸推动了回报？
- 前 3 大贡献者和前 3 大拖累者
- 是否有任何单一头寸的重大影响？

### 步骤 3：资产配置审查

当前配置与目标配置对比：

| 资产类别 | 目标 | 当前 | 偏离 | 行动 |
|------------|--------|---------|-------|--------|
| 美国大盘股 | | | | |
| 美国中/小盘股 | | | | |
| 国际发达市场 | | | | |
| 新兴市场 | | | | |
| 固定收益 | | | | |
| 另类投资 | | | | |
| 现金 | | | | |

标记任何超过投资政策声明（IPS）再平衡阈值（通常为 3-5%）的偏离。

### 步骤 4：讨论要点

生成会议议程：

1. **市场概览**（2-3 分钟）：简要宏观背景和展望
2. **投资组合业绩**（5 分钟）：我们表现如何？为什么？
3. **资产配置审查**（5 分钟）：是否需要再平衡？
4. **规划更新**（5-10 分钟）：
   - 生活变化？（工作、健康、家庭、住房、教育）
   - 收入需求是否变化？
   - 税务情况更新
   - 遗产规划更新
5. **行动项目**（5 分钟）：下次会议前我们要做什么？

### 步骤 5：主动建议

根据审查结果，建议：
- 再平衡交易（如果偏离超过阈值）
- 税收损失收割机会
- 现金部署或提取规划
- Roth 转换机会（如适用）
- 受益人更新或遗产规划需求
- 保险审查（人寿、残疾、长期护理）

### 步骤 6：输出

- 一页客户审查摘要（Word 或 PDF）
- 带有基准的业绩表
- 资产配置饼图（当前与目标）
- 建议的行动项目
- 会议议程

## 重要说明

- 会议前了解你的客户——审查上次会议的笔记
- 以客户关心的内容为主导，而不是你想谈论的内容
- 如果业绩不佳，直接面对——不要隐藏或粉饰
- 始终以明确的行动项目和带有日期的后续步骤结束
- 记录会议笔记和投资政策声明（IPS）的任何变更
- 合规：确保所有材料符合公司政策和监管要求
