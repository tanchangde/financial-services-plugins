# 财务规划

## 工作流程

### 步骤 1：客户档案

收集或确认：
- **人口统计**：年龄、配偶年龄、受抚养人、预期寿命假设
- **就业**：当前收入、预期加薪、目标退休年龄
- **账户**：所有投资账户及其余额和资产配置
- **收入来源**：工资、奖金、租金收入、社会保障金估算、养老金
- **支出**：当前年度支出、预期变化（抵押贷款还清、子女独立）
- **负债**：抵押贷款、学生贷款、其他债务
- **保险**：人寿、残疾、长期护理、健康
- **遗产**：遗嘱、信托、受益人指定、赠与策略

### 步骤 2：现金流分析

建立年度现金流预测：

| 年份 | 年龄 | 总收入 | 税款 | 生活支出 | 储蓄 | 净现金流 |
|------|-----|-------------|-------|-----------------|---------|--------------|
| | | | | | | |

关键输入：
- 通货膨胀率假设（通常为 2.5-3%）
- 税率（边际税率和有效税率）
- 储蓄率及储蓄去向（税前、Roth、应税）

### 步骤 3：退休预测

**积累阶段：**
- 当前投资组合价值
- 年度供款（401(k) 退休计划、个人退休账户（IRA）、应税账户）
- 按资产类别划分的预期回报
- 蒙特卡洛模拟（Monte Carlo）：各种支出水平的成功概率

**分配阶段：**
- 退休所需年度支出（今日美元 → 通胀调整）
- 社会保障金开始领取年龄和福利
- 养老金收入（如有）
- 投资组合提取率和顺序
- 强制最低提取额（RMDs）

**关键输出：**
- 退休时的预计投资组合价值
- 可持续提取率
- 资金不耗尽的概率（目标 >85%）
- "如果"情景：提前退休、市场低迷、更高支出

### 步骤 4：特定目标分析

#### 教育资金
- 子女年龄和目标大学入学时间
- 当前 529 计划余额
- 目标资金水平（公立与私立，4 年制与研究生）
- 达成目标所需的月度储蓄
- 经济援助考虑因素

#### 遗产规划
- 当前遗产价值和预计增长
- 遗产税风险（联邦和州）
- 现有信托结构
- 赠与策略（年度免税额、终身免税额使用）
- 慈善捐赠计划
- 受益人审查

#### 风险管理
- 人寿保险需求分析（收入替代、债务偿还、教育资金）
- 残疾保险充足性
- 长期护理规划
- 伞式责任保险

### 步骤 6：建议

优先行动项目：
1. 储蓄率变更
2. 资产配置调整
3. 税务优化（Roth 转换、税收损失收割、资产位置）
4. 填补保险缺口
5. 遗产文件更新
6. 受益人指定审查

### 步骤 7：输出

- 财务规划文件（Word/PDF，15-25 页）
- 现金流预测电子表格（Excel）
- 退休预测图表
- 目标资金分析
- 情景对比表
- 行动项目清单

## 重要说明

- 财务规划是动态文件——每年或重大生活事件后审查和更新
- 回报假设要保守——高估回报会带来虚假信心
- 税务规划与投资回报同样重要——为每项建议建模税务影响
- 社会保障金领取时机是重要杠杆——建模 62 岁、67 岁和 70 岁开始领取年龄
- 始终对规划进行压力测试——仅在基准情况下可行的规划不是好规划
- 合规：确保建议符合适当性/受托责任标准
