# 投资组合再平衡

## 工作流程

### 步骤 1：当前状态

针对每个账户，捕获：
- 账户类型（应税账户、个人退休账户（IRA）、Roth 账户、401(k) 退休计划）
- 当前市值的持仓
- 成本基础（针对应税账户）
- 每个头寸的未实现收益/损失

### 步骤 2：偏离分析

将当前配置与投资政策声明（IPS）目标进行对比：

| 资产类别 | 目标百分比 | 当前百分比 | 偏离 | 美元超/低配 |
|------------|----------|-----------|-------|-------------|
| 美国大盘股 | | | | |
| 美国中/小盘股 | | | | |
| 国际发达市场 | | | | |
| 新兴市场 | | | | |
| 投资级债券 | | | | |
| 高收益/信用债 | | | | |
| 通胀保值债券（TIPS） | | | | |
| 另类投资 | | | | |
| 现金 | | | | |

标记超过再平衡区间（通常为 ±3-5%）的头寸。

### 步骤 3：交易建议

生成交易以使配置回到目标：

**税务感知再平衡规则：**
- 优先在税收优惠账户（个人退休账户（IRA）、Roth 账户）中进行再平衡——无税务后果
- 在应税账户中，避免卖出具有大量短期收益的头寸
- 在再平衡的同时尽可能收割损失
- 在所有账户中注意洗售交易（wash sale）规则（30 天窗口）
- 考虑将新资金导向低配资产类别而非交易

**交易清单：**

| 账户 | 行动 | 证券 | 股数/美元 | 原因 | 税务影响 |
|---------|--------|----------|----------|--------|-----------|
| | 买入/卖出 | | | 再平衡 / 税收损失收割 | 短期收益 / 长期收益 / 损失 |

### 步骤 4：资产位置审查

优化哪些资产应持有哪些账户类型：
- **税收递延账户（个人退休账户（IRA）/401(k) 退休计划）**：债券、房地产投资信托基金（REITs）、高周转率基金（最高税务拖累）
- **Roth 账户**：预期增长最高的资产（免税增长）
- **应税账户**：税务高效的股票（指数基金、交易所交易基金（ETFs）、市政债券）、税收损失收割候选

### 步骤 5：实施

- 按账户汇总交易
- 估计交易成本
- 估计税务影响（已实现收益/损失）
- 对配置偏离的净影响

### 步骤 6：输出

- 偏离分析表
- 建议交易清单（Excel）
- 税务影响摘要
- 再平衡前后配置对比

## 重要说明

- 不要为了再平衡而再平衡——区间内的小幅偏离是可以的
- 在应税账户中，税务成本可能超过再平衡收益——计算盈亏平衡点
- 交易前考虑待处理的现金流（供款、提取、最低提取额（RMDs））
- 检查任何客户特定限制（环境、社会和治理（ESG）、集中持股、锁定期）
- 为合规记录记录每笔交易的理由
- 洗售交易（wash sale）规则适用于所有账户——协调家庭范围内的交易
