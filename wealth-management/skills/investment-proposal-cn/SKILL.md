# 投资建议书

## 工作流程

### 步骤 1：潜在客户背景

收集：
- **潜在客户姓名** 和家庭详情
- **当前情况**：现有顾问？自主投资？是什么促成了这次会议？
- **资产**：估计管理资产规模（AUM）、账户类型、当前持仓（如已分享）
- **目标**：退休、财富保值、增长、收入、教育、遗产
- **风险承受能力**：保守、稳健、激进（或问卷评分）
- **限制条件**：环境、社会和治理（ESG）偏好、集中持股、非流动性需求
- **费用敏感度**：他们现在支付多少？
- **竞争**：他们还在考虑哪些其他顾问？

### 步骤 2：建议书结构

**I. 关于我们公司**（1 页）
- 公司概览、历史、管理资产规模（AUM）
- 投资理念（用通俗英语）
- 团队简介（与该客户相关）
- 客户服务模式（多久见面一次，联系谁）

**II. 了解您的需求**（1 页）
- 重述他们的目标和担忧——表明您在倾听
- 发现过程中确定的关键规划考虑因素
- 对他们而言成功是什么样子的

**III. 拟议投资策略**（2-3 页）
- 建议的资产配置及理由
- 配置如何与他们的目标和风险承受能力相匹配
- 投资工具（交易所交易基金（ETFs）、共同基金、个人证券、另类投资）
- 税务意识策略（资产位置、税收损失收割）

拟议配置：

| 资产类别 | 配置 | 工具 | 理由 |
|------------|-----------|---------|-----------|
| | | | |

**IV. 预期结果**（1-2 页）
- 预计增长情景（保守、稳健、乐观）
- 蒙特卡洛模拟（Monte Carlo）达成目标的概率
- 收入预测（如退休或收入导向）
- 风险指标（最大回撤、波动率）
- 与当前投资组合的对比（如已知）

**V. 费用结构**（1 页）
- 咨询费用表（如适用，分级）
- 底层基金费用
- 总全包成本估算
- 费用与行业平均水平的对比
- 价值主张——他们支付费用获得什么

**VI. 开始合作**（1 页）
- 账户开设流程
- 资产转移时间线
- 过渡计划（如从其他顾问转移）
- 前 90 天——预期事项
- 所需文件和后续步骤

### 步骤 3：定制化

- 语气与潜在客户相匹配（企业高管 vs. 小企业主 vs. 退休人员）
- 如果他们持有集中股票头寸，直接解决
- 如果他们在将您与机器人顾问比较，强调规划和关系价值
- 如果他们对价格敏感，以总价值和结果为主导，而不仅仅是费用

### 步骤 4：输出

- 带有公司品牌的 PowerPoint 演示文稿（12-15 张幻灯片）
- PDF 可带走版本
- 用于后续电子邮件的一页摘要

## 重要说明

- 建议书应感觉个性化，而非模板化——引用他们的具体情况
- 不要过度推销业绩——设定期望并强调流程
- 始终包含免责声明（预测是假设性的、过往业绩等）
- 过渡计划很重要——客户担心更换顾问带来的干扰
- 在 48 小时内跟进建议书和明确的后续步骤
- 合规部门必须在向潜在客户展示前进行审查
