# 催化剂日历

## 工作流程

### 第一步：确定覆盖范围

- 要跟踪的公司列表（股票代码或名称）
- 行业/板块关注点
- 是否包含宏观事件？（美联储会议、经济数据、监管截止日期）
- 时间范围（未来2周、一个月、一个季度）

### 第二步：收集催化剂

为每家公司识别即将发生的事件：

**财报与财务事件**
- 季度财报日期和时间（盘前/盘后）
- 年度股东大会
- 投资者日/分析师日
- 资本市场日
- 债务到期/再融资日期

**公司事件**
- 产品发布或公告
- FDA批准/监管决策
- 合同续签或到期
- 并购里程碑（完成日期、监管批准）
- 管理层变动
- 内部人士交易窗口（锁定期到期）

**行业事件**
- 重要会议（日期、哪些公司演讲）
- 贸易展和博览会
- 监管意见征询期或裁决
- 行业数据发布（月度销售、流量等）

**宏观事件**
- 美联储会议（FOMC日期）
- 非农就业报告、CPI、GDP发布
- 央行决策（欧洲央行、日本央行等）
- 对市场有影响的地缘政治事件

### 第三步：日历视图

| 日期 | 事件 | 公司/板块 | 类型 | 影响程度(H/M/L) | 我们的持仓 | 备注 |
|------|------|----------|------|-----------------|------------|------|
| | | | 财报/公司/行业/宏观 | | 多头/空头/中性 | |

### 第四步：每周预览

每周生成前瞻性摘要：

**本周重点事件：**
1. [日期]：[公司]第[X]季度财报——共识预期[$X EPS]，我们预期[$X]，重点关注：[指标]
2. [日期]：[事件]——为什么这对[股票]重要
3. [日期]：[宏观发布]——预期和持仓建议

**下周预览：**
- 提前提醒即将到来的重要事件

**持仓影响：**
- 可能影响特定持仓的事件
- 建议的提前布局
- 重大事件前的风险管理

### 第五步：输出

- 日历视图和可排序列的Excel工作簿
- 每周预览邮件/笔记（markdown格式）
- 可选：与谷歌日历集成

## 重要说明

- 财报日期会变动——临近日期时通过公司投资者关系页面和彭博/FactSet核实
- 预公告风险：跟踪有预公告历史的公司（正面或负面）
- 会议出席名单很有价值——哪些公司在发言，哪些明显缺席？
- 有些催化剂是周期性的（月度行业数据）——建立模板并自动填充
- 按影响程度用颜色标识：红色=高影响，黄色=中等，绿色=常规
- 将过去的催化剂与实际结果一起归档——逐步建立模式识别
