# 财报预览 (Earnings Preview)

## 工作流程

### 步骤 1：收集背景信息

- 识别公司和报告季度
- 通过网络搜索获取一致预期估计（收入、每股收益、关键部门指标）
- 查找财报日期和时间（盘前 vs. 盘后）
- 回顾公司上季度财报电话会议的任何指引或评论

### 步骤 2：关键指标框架

为特定公司构建"关注点"框架：

**财务指标：**
- 收入 vs. 一致预期（总额和按部门）
- 每股收益 vs. 一致预期
- 利润率（毛利、营业、净利）——扩张还是收缩？
- 自由现金流
- 前瞻指引 vs. 一致预期

**运营指标**（特定于行业）：
- 科技/SaaS：年度经常性收入 (ARR)、净留存率、剩余履约义务 (RPO)、客户数量
- 零售：同店销售、客流、客单价
- 工业：积压订单、订单出货比、价格 vs. 销量
- 金融：净息差 (NIM)、信贷质量、贷款增长、手续费收入
- 医疗：处方量、患者量、管线更新

### 步骤 3：情景分析

构建 3 种情景及股价影响：

| 情景 | 收入 | 每股收益 | 关键驱动因素 | 股价反应 |
|------|------|----------|-------------|----------|
| 牛市 | | | | |
| 基准 | | | | |
| 熊市 | | | | |

对每种情景：
- 运营上需要发生什么
- 管理层评论将如何暗示
- 历史背景——股票在类似业绩时如何波动？

### 步骤 4：催化剂清单

识别将决定股票反应的 3-5 件事：

1. [指标] vs. [一致预期/传闻数字]——为何重要
2. [指引项目]——买方期望听到什么
3. [叙事转变]——任何战略变化、并购、重组

### 步骤 5：输出

一页财报预览，包含：
- 公司、季度、财报日期
- 一致预期估计表格
- 关键关注指标（按重要性排序）
- 牛/基准/熊情景表格
- 催化剂清单
- 交易布局：近期股票表现、期权隐含波动

## 重要说明

- 一致预期估计会变化——始终注明估计来源和日期
- 买方调查的"市场传闻预期"（Whisper number）通常比公布的一致预期更相关
- 历史财报反应有助于校准预期（搜索"[公司] 财报反应历史"）
- 期权隐含波动幅度告诉市场期望什么——与你的情景比较
