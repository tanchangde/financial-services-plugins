# 模型更新 (Model Update)

## 工作流程

### 步骤 1：识别变化内容

确定更新触发因素：
- **财报发布**：新的季度实际数据填入
- **指引变化**：公司更新前瞻展望
- **估计修订**：分析师基于新数据改变假设
- **宏观更新**：利率、外汇、商品价格变化
- **事件驱动**：并购、重组、新产品、管理层变更

### 步骤 2：填入新数据

#### 财报后
用报告的实际数据更新模型：

| 项目 | 前期估计 | 实际 | 变动 | 备注 |
|------|----------|------|------|------|
| 收入 | | | | |
| 毛利率 | | | | |
| 营业费用 | | | | |
| 息税折旧摊销前利润 | | | | |
| 每股收益 | | | | |
| [关键指标 1] | | | | |
| [关键指标 2] | | | | |

**部门明细**（如适用）：
- 更新每个部门的收入和利润率
- 注意任何部门组合变化

**资产负债表/现金流量表更新**：
- 现金和债务余额
- 股数（回购、稀释）
- 资本支出实际 vs. 估计
- 营运资本变化

### 步骤 3：修订前瞻估计

基于新数据，调整前瞻估计：

| | 旧财年估计 | 新财年估计 | 变化 | 旧下一财年 | 新下一财年 | 变化 |
|---|-----------|-----------|------|-----------|-----------|------|
| 收入 | | | | | | |
| 息税折旧摊销前利润 | | | | | | |
| 每股收益 | | | | | | |

**关键假设变化：**
- 正在改变什么假设及原因？
- 收入增长率：旧 → 新（原因）
- 利润率假设：旧 → 新（原因）
- 任何新项目（重组费用、一次性收益等）

### 步骤 4：估值影响

用更新后估计重新计算估值：

| 估值方法 | 前期 | 更新后 | 变化 |
|----------|------|--------|------|
| DCF 公允价值 | | | |
| 市盈率 (远期每股收益 × 目标倍数) | | | |
| 企业价值/息税折旧摊销前利润 (远期 × 目标倍数) | | | |
| **价格目标** | | | |

### 步骤 5：摘要与行动

**估计变化摘要：**
- 一段话：什么变化、为什么、对股票意味着什么
- 这是改变投资论点的事件还是噪音？

**评级/价格目标：**
- 维持或改变评级？
- 新价格目标（如变化）及方法
- 相对当前价格的上涨/下跌空间

### 步骤 6：输出

- 更新后 Excel 模型（如用户提供现有模型）
- 估计变化摘要（markdown 或 Word）
- 更新后价格目标推导

## 重要说明

- 预测前始终将估计与公司报告数据核对
- 注意任何非经常性项目，估计是 GAAP 还是调整后
- 跟踪估计修订历史——显示分析进展
- 如季度波动大，在估计变化中区分信号与噪音
- 更新后检查一致预期——修订后估计与市场一致预期比较如何？
- 股数很重要——来自股票薪酬、可转债或回购的稀释可能实质性影响每股收益
