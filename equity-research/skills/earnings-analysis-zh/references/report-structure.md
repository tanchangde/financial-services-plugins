# 报告结构和模板 (Report Structure and Templates)

本文档提供财报更新 DOCX 报告的完整逐页模板和格式要求。

## 第 1 页：财报摘要 (EARNINGS SUMMARY)

**顶部区域 - 页眉：**
```
[公司名称] ([股票代码])
[季度] [年份] 财报更新

[当前日期]

评级：[维持/上调/下调] [评级]
股价（截至 [日期]）：XX.XX 美元
价格目标：[如变化则写 旧 → 新，或维持 XXX 美元]
```

**顶部区域 - 快速摘要框：**
```
财报摘要
─────────────────────────────────────────────────
2024 年第 [X] 季度业绩：[超预期 / 符合预期 / 低于预期]

                报告业绩    估计      差异
收入            X,XXX       X,XXX     +$XXX (+X%)
每股收益（调整后） X.XX       X.XX      +$X.XX (+X%)

关键要点：
■ [要点 1 - 一句话]
■ [要点 2 - 一句话]
■ [要点 3 - 一句话]
```

**主要内容 - 投资影响（3-4 个要点）：**

使用 ■ 字符和**粗体标题**及段落长度解释：

```
■ **业绩超预期，得益于 [部门/地理/产品] 强劲，保持积极势头**

第 [X] 季度收入 X.X 十亿美元超出我们 X.X 十亿美元估计 X%，超一致预期 X%，
主要由 [具体驱动因素] 推动。[部门] 收入同比增长 X%（我们估计 X%），
而 [部门] 增长 X%（我们估计 X%）。管理层强调 [具体产品/计划] 为关键
增长驱动因素，并对前景保持信心。超预期证明了 [投资论点要点]，强化
了我们的积极观点。

■ **利润率同比扩张 XX 个基点，尽管 [逆风]，展示经营杠杆**

[详细利润率分析段落...]

■ **指引上调 / 维持 / 下调——意味着 [解读]**

[详细指引分析段落...]

■ **维持 [评级]，[上调/不变] XXX 美元价格目标**

[投资结论段落...]
```

## 格式要求

### 1. 第 1 页要求
- 清晰评级（维持增持、上调至买入等）
- 醒目显示更新后价格目标
- 摘要表格含旧/新估计
- 3-4 个带 ■ 字符的段落长度要点

### 2. 所有表格要求
- 底部有来源行
- 清晰列标题
- 标题行底色

### 3. 所有图表要求
- 上方有"图 X - [标题]"说明
- 下方有"来源：[来源]"行
- 专业样式

### 4. 年度标注
- 使用 A 表示实际（Q3'24A）
- 使用 E 表示估计（Q4'24E）

### 5. 写作风格
- 数据先行（"收入增长 15% 至 12 亿美元"而非"强劲收入增长"）
- 使用"vs."而非"versus"
- 直接简洁
- 聚焦新信息

### 6. 超链接要求 ⭐⭐⭐
- Word 中所有 URL 必须是可点击超链接
- 蓝色、带下划线文本，Ctrl+Click 打开
- 显示文本有意义（非原始 URL）
- 每个来源引用在适用时应有可点击链接
- 无纯文本 URL——始终格式化为超链接
