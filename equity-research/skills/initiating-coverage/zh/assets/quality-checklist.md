# 首次覆盖报告质量检查清单 (30-50页文档)

本清单确保首次覆盖报告达到机构级（摩根大通/高盛/摩根士丹利水平）。

## 1. 核心投资摘要 (核心内容)

- [ ] **投资要点**：是否明确阐述了“为何现在买入/卖出”？
- [ ] **目标价说明**：是否明确说明了目标价、当前价格及涨跌幅？
- [ ] **评级**：是否已有明确评级（买入/中性/卖出）？
- [ ] **关键催化剂**：是否列出了未来 6-12 个月内的 3-5 个具体催化剂？
- [ ] **风险因素**：是否突出了前 3 大风险，而非仅仅列举？

## 2. 公司业务分析 (6,000-8,000字)

- [ ] **业务细分**：是否详细分析了每个产品线/业务部门？
- [ ] **管理层**：是否为至少 3 名高管提供了 300-400 字的详细简历？
- [ ] **市场定位**：是否清晰界定了 TAM（潜在市场总量）？
- [ ] **竞争对手**：是否对 5-10 家主要竞争对手进行了深入对比分析？
- [ ] **护城河分析**：是否明确说明了公司的竞争优势？

## 3. 财务模型与预测 (10-15页)

- [ ] **收入模型**：是否包含产品/地域维度的详细预测（20-30行）？
- [ ] **利润率趋势**：是否详细解释了毛利率和经营利润率的变化趋势？
- [ ] **现金流**：自由现金流 (FCF) 的预测是否逻辑严密？
- [ ] **情景分析**：牛市/基准/熊市情景是否具有显著差异且逻辑合理？
- [ ] **资本配置**：是否讨论了分红、回购和并购策略？

## 4. 估值分析 (5-10页)

- [ ] **DCF 分析**：
    - [ ] 是否包含 5 年详细预测 + 终值？
    - [ ] WACC 计算是否透明并说明了假设？
    - [ ] 是否包含 WACC 与终值增长率的敏感性矩阵？
- [ ] **可比公司 (Comps)**：
    - [ ] 是否分析了 5-10 家同行？
    - [ ] 是否包含了 P/E, EV/EBITDA, P/S 等多重估值倍数？
    - [ ] 是否包含了统计摘要（中位数、平均值、分位数值）？
- [ ] **估值足球场图**：是否直观展示了不同估值方法的区间？

## 5. 视觉与排版要求 (25-35张图表)

- [ ] **图表密度**：每 200-300 字是否配有一张图表/表格？
- [ ] **图表质量**：所有图表是否均为 300 DPI 且格式一致？
- [ ] **强制性图表**：
    - [ ] 按产品划分的收入 (堆积面积图)？
    - [ ] 按地域划分的收入 (堆积柱状图)？
    - [ ] DCF 敏感性分析 (二维热力图)？
    - [ ] 估值足球场图？
- [ ] **表格格式**：所有财务表格是否对齐且易于阅读？

## 6. 报告结构与导航

- [ ] **目录**：目录是否包含跳转到各章节的超链接？
- [ ] **脚注与引用**：所有数据来源是否通过超链接正确引用？
- [ ] **免责声明**：是否包含标准的机构法律免责声明？
- [ ] **报告长度**：总长度是否在 30-50 页之间？
- [ ] **字数**：总字数是否超过 10,000 字？

## 7. 最终审阅

- [ ] **逻辑一致性**：正文叙述、财务模型和图表中的数字是否完全一致？
- [ ] **语调**：是否保持了客观、专业、不带偏见的机构研究语调？
- [ ] **校对**：是否已消除拼写和语法错误？
- [ ] **PDF 生成**：DOCX 导出为 PDF 后格式是否依然完美？

---
**目标**：如果以上所有项均勾选，则报告已具备向机构投资者发布的水平。
