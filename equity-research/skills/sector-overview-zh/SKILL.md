# 板块概述 (Sector Overview)

## 工作流程

### 步骤 1：定义范围

- **板块/子板块**：什么行业及定义多细？
- **目的**：客户报告、内部研究、推介材料、想法生成
- **深度**：高级概述（5-10 页）或深度分析（20-30 页）
- **角度**：中性景观 vs. 主题投资论点（例如"AI 基础设施建设"）
- **范围**：仅上市公司，或包括私营公司？

### 步骤 2：市场概述

**市场规模与增长**
- 总可寻址市场 (TAM)，附来源
- 历史增长率（5 年复合年增长率）
- 预测增长率和关键假设
- 市场细分（按产品、地理、终端市场、客户类型）

**行业结构**
- 分散 vs. 集中——前 5 大市场份额
- 价值链地图——价值在哪里积累？
- 商业模式类型（订阅、交易、授权、服务）
- 进入壁垒（资本、监管、技术、网络效应）

**关键趋势与驱动因素**
- 长期结构性顺风因素（3-5 个主要趋势）
- 逆风因素和阻力
- 技术颠覆方向
- 监管发展
- 并购活动和整合趋势

### 步骤 3：竞争格局

**公司简介**（前 5-10 家参与者）：

| 公司 | 收入 | 增长 | 息税折旧摊销前利润率 | 市场份额 | 关键差异化因素 |
|------|------|------|---------------------|----------|---------------|
| | | | | | |

对每个公司，简要简介：
- 业务描述（2-3 句话）
- 战略定位和护城河
- 近期发展（财报、并购、产品发布）
- 估值快照（市盈率、企业价值/息税折旧摊销前利润、企业价值/收入）

**竞争动态**
- 公司如何竞争？（价格、产品、服务、分销）
- 谁在获得/失去份额及原因？
- 来自新进入者或相邻参与者的颠覆风险

### 步骤 4：估值背景

- 板块交易倍数（当前和历史区间）
- 溢价/折价驱动因素（增长、利润率、市场地位）
- 近期并购交易倍数
- 板块与更广泛市场比较如何？

### 步骤 5：投资影响

- 哪里是最好的风险/回报机会？
- 通过这个板块可以表达什么主题押注？
- 板块的关键辩论（牛市 vs. 熊市论点）
- 可能改变板块叙事的催化剂

### 步骤 6：输出

- Word 文档或 PowerPoint，包含：
  - 市场概述和规模
  - 竞争格局图
  - 公司比较表格
  - 估值摘要
  - 关键图表：市场增长、份额趋势、估值历史
- Excel 附录含详细公司数据

## 重要说明

- 引用所有市场规模数据——注明研究公司或方法
- 区分 TAM 炒作和现实可寻址市场
- 板块概述快速过时——注明日期并标记可能过时的数据
- 图表必不可少——市场规模瀑布图、竞争定位矩阵、估值散点图
- 如为客户，根据具体情况定制"投资启示"（并购目标识别、竞争定位、市场进入）
