# 想法生成 (Idea Generation)

## 工作流程

### 步骤 1：定义搜索标准

询问用户参数：
- **方向**：多头想法、空头想法或两者
- **市值**：大型、中型、小型、微型
- **行业**：特定行业或跨行业
- **风格**：价值、成长、质量、特殊情况、事件驱动
- **地理**：美国、国际、全球
- **主题**：任何特定主题角度（AI、回流、人口老龄化等）

### 步骤 2：定量筛选

根据风格运行筛选：

**价值筛选**
- 市盈率低于行业中位数
- 企业价值/息税折旧摊销前利润低于历史平均水平
- 自由现金流收益率 >5%
- 市净率低于 1.5 倍
- 过去 90 天内有内部人买入
- 股息收益率高于市场平均水平

**成长筛选**
- 收入同比增长 >15%
- 收益同比增长 >20%
- 收入加速（增长率上升）
- 利润率扩张
- 高投入资本回报率 (>15%)
- 强净留存率（SaaS >110%）

**质量筛选**
- 一致的收入增长（5 年以上）
- 稳定或扩张的利润率
- 净资产收益率 >15%
- 低债务/股本
- 高自由现金流转化率
- 内部人持股 >5%

**空头筛选**
- 收入下降或增长减速
- 利润率压缩
- 应收账款/库存相对销售上升
- 内部人卖出
- 相对同行估值溢价但无理由
- 高做空兴趣但基本面恶化
- 会计危险信号（审计师变更、重述）

**特殊情况筛选**
- 近期 IPO/SPAC 且锁定期到期
- 过去 12 个月内的分拆
- 从重组中恢复的公司
- 激进投资者介入
- 表现不佳公司的管理层变更

### 步骤 3：主题扫描

对于主题想法，研究主题并识别受益者：

1. 定义论点（例如"AI 基础设施支出在 2026 年前加速"）
2. 绘制价值链——谁直接受益 vs. 间接受益？
3. 识别纯标的 vs. 多元化敞口
4. 评估哪些标的已被"定价"vs. 被低估
5. 寻找市场尚未与该主题关联的第二层受益者

### 步骤 4：想法呈现

对于通过筛选的每个想法，呈现：

**[公司名称] — [多头/空头] — [一句话投资论点]**

| 指标 | 值 | vs. 同行 |
|------|-----|----------|
| 市值 | | |
| 企业价值/息税折旧摊销前利润 (远期) | | |
| 市盈率 (远期) | | |
| 收入增长 | | |
| 息税折旧摊销前利润率 | | |
| 自由现金流收益率 | | |

**投资论点（3-5 个要点）：**
- 为何被错误定价
- 市场忽略了什么
- 实现价值的催化剂

**关键风险：**
- 什么会使这个想法错误

**建议的下一步：**
- 建立完整模型？深入尽职调查？专家电话？

### 步骤 5：输出

- 5-10 个想法的简短列表，含一页摘要
- 筛选标准和方法记录
- 所有想法的比较表格
- 优先级列表：哪些想法优先研究

## 重要说明

- 筛选表面候选者，非结论——每个筛选输出需要基本面工作
- 最好的想法通常来自交叉点（例如因暂时逆风而以价值价格交易的质量公司）
- 避免拥挤交易——检查持股数据、做空兴趣和覆盖该公司的分析师数量
- 反向想法需要催化剂——没有催化剂的早期等同于错误
- 跟踪想法命中率——哪些筛选和方法产生最好的想法？
- 空头想法需要更高信念——时机更难且风险不对称
